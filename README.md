# Omar Osama CV

This is a personal CV webpage of Omar Osama, a Computer Science student and Junior Frontend Developer.  
The project consists of a single HTML file showcasing my CV.

---

## Features

- Semantic HTML structure
- SEO meta tags (title, description, author, keywords)
- Open Graph (OG) tags for social media sharing
- Favicon included
- Sections: General Info, Skills, Education, Experience, Contact Links

---

## How to View

1. Download or clone this repository
2. Open the `index.html` file in any web browser
3. The CV webpage will display
